Comandos Git

Git init -> para iniciar um novo repositorio

Git add + nome do arquivo -> para subir o arquivo para o Stage

Git rm --cached + nome do arquivo -> para remover alteracoes feitas no arquivo do Stage

Git commit -m "Nome do arquivo" -> para subir o arquivo no GitHub

Git log -> para listar os commits feitos no repo

Git checkout -> para navegar nos commits

Git pull -> puxa modificacoes do repositorio para o local

Git push -> envia as modificacoes do repositorio local para o remoto

Git clone +"Caminho" -> para clonar um repositorio do git
